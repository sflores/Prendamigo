QADefect Client Example
===============================================================================

Description 
===============================================================================
- a simple example that introduces Wink Client usage
- demonstrate the consumption of an Atom feed of defects from the QADefect
  service example which can be found at SDK_HOME/examples/apps/QADefect

Build
===============================================================================
- use Ant version 1.7.0 or greater
- build the example using 'ant'
- an executable jar is created under the example directory  

Running the Example
===============================================================================
- make sure that the QADefect example is up and running on a local server
- to run the example, run the following command: java -jar QADefects-client.jar
