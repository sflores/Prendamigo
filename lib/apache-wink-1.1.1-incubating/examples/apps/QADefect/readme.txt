QADefect Example
===============================================================================

Description 
===============================================================================
- Advanced implementation of a QA defect web service showing
  a custom implementation of resources and assets
- Defects can be listed, created, updated, and deleted
- Demonstrated usage of Atom, CSV, HTML, OpenSearch and XML via JAXB
- Demonstrated usage of defects' attachment binary representation
- The list of all defects is returned as an Atom feed on this URI:
  http://localhost:8080/QADefect/rest/defects

-Wink features used in this example:
   Asset, ServiceDocument, LinkBuilders, SyndFeed, SyndEntry, Html Representation
