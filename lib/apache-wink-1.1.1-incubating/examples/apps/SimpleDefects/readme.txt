SimpleDefects Example
===============================================================================

Description 
===============================================================================
- Simple implementation of a QA defect application demonstrating
  a custom implementation of resource and asset classes
- Atom, JSON and XML via JAXB usage is demonstrated
- Defects can be listed, created, updated, and deleted
- The list of all defects is returned as an Atom feed on this URI:
  http://localhost:8080/SimpleDefects/rest/defects

- Wink features used in this example:
   ServiceDocument, Asset, LinkBuilders, SyndFeed, SyndEntry
